This library contains a shared interface to let you write a plugin for [Fail2Ban4Win](https://github.com/Aldaviva/Fail2Ban4Win).

Your plugin project should depend on this package.

- [Documentation with examples](https://github.com/Aldaviva/Fail2Ban4Win/wiki/Plugin-Authoring)
- [Sample plugin source code](https://github.com/Aldaviva/Fail2Ban4Win/tree/master/SamplePlugin)
